# News Analyzer & Multilingual Podcast Generator

## Overview

This project analyzes news articles, extracts keywords, generates summaries, translates the summaries into multiple languages, and creates audio podcasts using text-to-speech.

## Features

* News text cleaning
* Keyword extraction
* Category detection
* Automatic summarization
* English, Telugu, and Hindi translation
* Multilingual podcast generation
* Report generation

## Technologies Used

* Python
* gTTS
* deep-translator

## Project Structure

* src/ : Source code
* data/ : Input news files
* output/ : Generated reports and podcasts
